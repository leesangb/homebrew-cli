# homebrew-cli

Homebrew tap for `leesangb` command-line tools.

## Install

```bash
brew install leesangb/cli/wt
```

Or tap once and install by formula name:

```bash
brew tap leesangb/cli
brew install wt
```

## Shell Integration

`wt` can install optional shell integration for `bash`, `zsh`, and `fish`:

```bash
wt shell install zsh
```

Then run the command that `wt` prints to append the integration to your shell config.

Upgrade with Homebrew:

```bash
brew upgrade wt
```
